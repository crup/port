# Support

Use the right channel so issues are actionable:

- Bug reports: GitHub Issues
- Feature requests: GitHub Issues
- Security reports: [`SECURITY.md`](SECURITY.md)
- Contribution questions: [`CONTRIBUTING.md`](CONTRIBUTING.md)

When opening an issue, include:

- browser and runtime details
- host origin and child origin setup
- whether the issue happens in inline mode, modal mode, or both
- a minimal reproduction when possible
