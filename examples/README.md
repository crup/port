# Examples

These examples stay intentionally small and map directly to the library API.

- `host-inline.ts`: mount an iframe inline and listen for child events
- `host-modal.ts`: mount a modal port and open it explicitly
- `child-basic.ts`: respond to host requests and emit resize signals

For a polished live demo with a real host and child page, use:

- local dev: `npm run demo:dev`
- production demo: https://crup.github.io/port/
