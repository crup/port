# Code Of Conduct

## Our Standard

This project adopts the Contributor Covenant Code of Conduct.

In this community, contributors are expected to:

- be respectful in discussion and review
- focus on technical clarity over personal attacks
- assume good faith while still challenging weak ideas directly
- help keep the project usable for maintainers and consumers

## Unacceptable Behavior

- harassment or intimidation
- insulting or demeaning language
- publishing private information without permission
- disruptive behavior that makes collaboration materially harder

## Enforcement

Report conduct issues to `connect@rajender.pro`.

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this policy.

## Scope

This code of conduct applies in project spaces and when someone is representing the project in public spaces.
